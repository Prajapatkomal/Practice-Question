Q. What is useMemo Hook? How does useMemo help in performance optimization?

Ans- useMemo is a React Hook used to memoize the speific value or result of an expensive calculation within a functional component.
It takes two arguments: a function that returns the value to be memoized, and an array of dependencies. The function will only re-run if any of the dependencies in the array have changed.

It returns the memoized value: const memoizedValue = useMemo(() => expensiveCalculation(a, b), [a, b]);
